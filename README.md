# products

This is a demo VueJs3 project, obviously overkill.
WIP - no css yet.
Purpose is to illustrate :

- Application structure
- Router
- API call
- Pina store
- watch, computed

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
